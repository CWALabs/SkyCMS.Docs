# Storage Configuration Reference

## Core settings

- provider type,
- container/bucket name,
- access credentials,
- endpoint and region settings where applicable.

## Provider fields

### Azure Blob

- connection string,
- container name.

### S3-compatible

- access key ID,
- secret access key,
- bucket,
- region,
- optional endpoint override.

### Cloudflare R2

- endpoint,
- key ID/secret,
- bucket name.

### Google Cloud Storage

- bucket,
- service account credentials,
- project context.
