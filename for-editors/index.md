# For Editors

## When to use this page

Use this page when you need to:

- understand editor responsibilities,
- find lifecycle and operational workflows quickly,
- navigate to the right editing tool guide.

## Role Overview

SkyCMS content roles work together like this:

- Editors create, revise, schedule, publish, unpublish, and retire content.
- Site builders define how content is laid out and presented.
- Developers extend platform behavior, integrations, and infrastructure.

In this section, an editor is the person responsible for day-to-day content operations.

Editors typically:

- Create and update articles and pages.
- Keep content in draft until it is ready.
- Publish immediately or schedule publication.
- Unpublish or delete outdated content.
- Validate the live result after changes.
