# Visual Editor Quick Start

## Summary

Use this guide to make your first content update with the SkyCMS Visual Editor.

If you are new to SkyCMS, this is the fastest path from opening a page to publishing an update safely.

## Outcome

After this quick start, you will be able to:

- open a page in edit mode,
- format text and insert media,
- save and publish changes,
- handle common editing issues.

## Prerequisites

- You can sign in to SkyCMS.
- You have editor permissions for at least one page.

## Edit flow

```mermaid
flowchart LR
  Open[Open page] --> Edit[Enter edit mode]
  Edit --> Update[Update content]
  Update --> Save[Save changes]
  Save --> Publish[Publish now or schedule]
```

## Steps

1. Open **Editor** and choose a page.
1. Click **Edit**.
1. Click inside an editable region.
1. Use the inline toolbar to format content.
1. Save changes.
1. Publish now or schedule publication.

## Common tasks

### Add text formatting

- Select text.
- Apply formatting such as bold, italic, heading, or list style.

### Insert an image

1. Choose the image insert action.
1. Upload a new image or select existing media.
1. Add alt text and adjust display size.

### Insert an internal page link

1. Select the text to link.
1. Open the page link tool.
1. Choose the destination page and apply.

## Save and publish notes

- Auto-save runs after short idle periods.
- Manual save shortcut: `Ctrl+S` or `Cmd+S`.
- Choose publish now for immediate updates.
- Use scheduling when content should go live later.

## Troubleshooting basics

- Toolbar missing: click inside an editable region again.
- Save issue: retry manual save and verify network connectivity.
- Upload issue: verify file type and file size policy.
