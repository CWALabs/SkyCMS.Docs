# Visual Editor Quick Start

## Getting Started

1. Open **Editor** and choose a page.
2. Click **Edit**.
3. Click inside an editable region.
4. Use the inline toolbar for formatting.

## Common Tasks

### Add Text Formatting

- Select text and choose bold/italic/heading/list actions.

### Insert Image

1. Select image insert action.
2. Upload or choose existing media.
3. Add alt text and adjust size.

### Insert Internal Page Link

1. Select text.
2. Open page link tool.
3. Pick target page and apply.

## Save and Publish

- Auto-save runs after short idle periods.
- Manual save: `Ctrl+S` / `Cmd+S`.
- Use publish now or publish later depending on release needs.

## Troubleshooting Basics

- Toolbar missing: click back into editable region.
- Save issue: retry manual save and check connectivity.
- Upload issue: verify file format and size.
