# Page Builder Quick Start

## Summary

Use this guide to create a basic layout or template and publish a page with Page Builder.

If you are new to SkyCMS, this is the fastest visual path to assemble content without writing code.

## Outcome

After this quick start, you will be able to:

- place blocks in a structured page layout,
- style content visually,
- save, preview, and publish your page.

## Prerequisites

- You can sign in to SkyCMS.
- You have permissions to edit pages, templates, or layouts.

## Builder flow

```mermaid
flowchart LR
  Compose[Compose blocks] --> Style[Apply visual styles]
  Style --> Preview[Preview responsive result]
  Preview --> Save[Save changes]
  Save --> Publish[Publish page]
```

## Steps

### Build a basic layout

1. Open **Layouts** and start Page Builder.
1. Add a header/navigation block to the top region.
1. Add a footer block to the bottom region.
1. Adjust spacing and typography in the style manager.
1. Save and preview.

### Build a basic template

1. Open **Templates** and start Page Builder.
1. Add a section and a content container.
1. Add heading, text, and button blocks.
1. Adjust colors and spacing.
1. Save the template.

### Build and publish a page

1. Open **Editor** and select Page Builder.
1. Add text and media blocks.
1. Apply visual styles.
1. Save and preview.
1. Publish when ready.

## Essential tips

- Use the Layers panel to manage nested elements clearly.
- Save frequently during larger edits.
- Check tablet and mobile preview before publishing.
- Keep block structure simple so the page stays easier to maintain over time.

## Keyboard basics

- Save: `Ctrl+S` or `Cmd+S`
- Undo: `Ctrl+Z` or `Cmd+Z`
- Redo: `Ctrl+Shift+Z` or `Cmd+Shift+Z`
