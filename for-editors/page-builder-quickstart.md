# Page Builder Quick Start

## 5-Minute Start

### Build a Basic Layout

1. Open **Layouts** and start Page Builder.
2. Drag a navigation/header block into the top region.
3. Drag a footer block into the bottom region.
4. Style spacing and typography in the style manager.
5. Save and preview.

### Build a Basic Template

1. Open **Templates** and start Page Builder.
2. Add a section and container.
3. Add heading, text, and button blocks.
4. Adjust colors/spacing.
5. Save the template.

### Build a Page Quickly

1. Open **Editor** and choose Page Builder.
2. Add text and media blocks.
3. Apply visual styles.
4. Save, preview, publish.

## Essential Tips

- Use the layers panel to manage nested elements.
- Save frequently.
- Check tablet/mobile view before publishing.
- Keep block structure simple for maintainability.

## Keyboard Basics

- Save: `Ctrl+S` / `Cmd+S`
- Undo: `Ctrl+Z` / `Cmd+Z`
- Redo: `Ctrl+Shift+Z` / `Cmd+Shift+Z`
