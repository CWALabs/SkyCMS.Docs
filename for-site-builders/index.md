# For Site Builders

## Summary

Use this page when you need to:

- understand site builder responsibilities,
- find the right guide for layouts, templates, pages, or example implementations,
- navigate quickly to the supporting implementation guides.

## Role overview

SkyCMS content roles work together like this:

- Editors manage content operations and publishing.
- Site builders shape templates, layouts, and page composition.
- Developers implement deeper platform and integration changes.

In this section, a site builder is the person responsible for how content is structured and presented to end users.

Site builders typically:

- Design and maintain templates and reusable page structures.
- Define layout behavior and visual composition rules.
- Ensure pages are responsive, consistent, and brand-aligned.
- Collaborate with editors on authoring patterns and handoff.
- Coordinate with developers when implementation changes are required.
