# Widgets

Developer docs for SkyCMS widgets. Each widget provides focused, reusable UI behavior used by the Editor and runtime views.

- [Image Widget](./image.md) — upload/replace images with FilePond, manage alt text/title, browse library.
- [Crypto Widget](./crypto.md) — simple AES encrypt/decrypt helpers backed by CryptoJS.
- [Crumbs Widget](./breadcrumbs.md) — build breadcrumb navigation from meta path and title.
- [Forms Helpers](./forms.md) — post forms with antiforgery token and utilities.
- [Nav Builder](./nav-builder.md) — render a nav list using the ToC API.
- [Search Widget](./search.md) — basic search form wiring and result rendering.
- [ToC Widget](./table-of-contents.md) — build table-of-contents lists and breadcrumbs.

If you add new widgets, list them here and link their docs.
