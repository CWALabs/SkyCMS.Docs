# Setup Wizard: Step 2 Admin Account

## Summary

Create the initial administrator account used to manage SkyCMS.

## Outcome

After completing this step, the first administrator account will be created and ready to sign in after setup finishes.

## Required fields

- Admin email
- Password
- Password confirmation

## Password guidance

Use a strong unique password with uppercase, lowercase, numeric, and symbol characters.

## What to decide here

Use an email address that your team controls long term. Treat this account as the bootstrap administrator and plan to add named admin accounts after setup instead of sharing one credential.

## Common issues

- Password complexity not met.
- Password confirmation mismatch.
- Email already exists.

## Verification

This step is complete when the wizard accepts the administrator details and allows you to continue to publisher settings.
