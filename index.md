# SkyCMS Documentation

**Fast, edge-native CMS for developers and content teams.**

SkyCMS is a multi-tenant ASP.NET Core content management system that combines visual editing, flexible publishing, and edge-native deployment. Build and manage websites with multiple editors, publish to any cloud provider, and serve content from the edge.
